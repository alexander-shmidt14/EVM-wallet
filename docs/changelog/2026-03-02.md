# 2026-03-02 — Ретроспективная запись

**Раздел:** [[changelog/_index|Changelog]] · **Главная:** [[_index]]

---

> Эта запись покрывает всю историю проекта до создания документации.

---

## Phase 1 — Инициализация

- Создание монорепо (pnpm workspaces)
- Базовая структура: `apps/desktop`, `apps/mobile`, `packages/wallet-core`, `packages/ui-tokens`
- Настройка TypeScript 5.9, React 18, Electron 25
- Первый рабочий прототип кошелька

## Phase 2 — Wallet Core

- `WalletCore` класс: seed create/import, ETH balance, send ETH
- ERC-20 поддержка: `getErc20Balance`, `sendErc20`, `getErc20Info`
- MMA токен (`meta.ts`): адрес, decimals, символ
- Secure Store: AES-256-GCM шифрование seed через `safeStorage`
- Мульти-кошелёк: `listSeeds`, `deleteSeed`, `renameSeed`

## Phase 3 — Desktop Frontend

- 8 экранов: SetPassword → Login → WalletSelect → CreateWallet → ImportWallet → Wallet → Send → Receive
- Zustand store с IPC-маппингом
- ErrorBoundary, Loading, QRCode, Toast компоненты
- HashRouter навигация с auth guard

## Phase 4 — DevOps & Infrastructure

- GitHub: `alexander-shmidt14/EVM-wallet` (private)
- 3-ветвочная стратегия: `dev` → `staging` → `main`
- CI Pipeline (PR gate): typecheck + test + build
- Windows Build Pipeline: автоматическая сборка .exe
  - Electron cache (YAML) — ускорение на 30–60 сек
  - PowerShell retry (3 попытки) — борьба с flaky downloads
- Android Pipeline (disabled, manual only)
- Docker Compose: dev, test, typecheck, build сервисы
- `electron-builder-nosign.json` — unsigned builds для CI

## Phase 5 — Frontend/Backend разделение

- PR #3: `src/main.ts` разделён на `src/backend/main.ts`
- Теперь main process ≠ renderer entry

## Phase 6 — UI Polish

- Feature `fix/UI-titles`: иконка `icon_app.PNG` на LoginScreen
- Feature `fix/token-icon`: SVG компоненты `EthIcon.tsx` и `MmaIcon.tsx`
  - Заменили текстовые плейсхолдеры иконок на WalletScreen

## Phase 7 — Документация

- Obsidian vault в `docs/`
- ~45 страниц: architecture, backend, frontend, packages, devops, guides, changelog
- MOC-паттерн, `[[wikilinks]]`, YAML frontmatter
- Mermaid-диаграммы для data flow, CI, навигации
