# 2026-03-04 — Auto-Update Filename Fix

**Раздел:** [[changelog/_index|Changelog]] · **Главная:** [[_index]]

---

## Исправлено (Fixed)

### Auto-Updater: имя installer-а
- **🔧 `artifactName`**: добавлен явный `"artifactName": "EVM-Wallet-Setup-${version}.exe"` в блок `"win"` обоих конфигов:
  - `apps/desktop/electron-builder-nosign.json` (CI builds)
  - `apps/desktop/package.json` → `"build"."win"` (local builds)
- **Причина**: без `artifactName` electron-builder генерировал имя `EVM Wallet Setup X.Y.Z.exe` (с пробелами из `productName`), а `latest.yml` ссылался на это же имя. При скачивании обновления пробелы вызывали URL-encoding mismatch и updater не мог найти файл. Теперь имя без пробелов (`EVM-Wallet-Setup-X.Y.Z.exe`) — `latest.yml` и реальный файл `.exe` всегда совпадают.
- **🔧 `installer.nsi`**: обновлён `OutFile` для консистентности с новым именем
